## ✅ 關鍵步驟檢查清單

### 準備階段
- [ ] 有 GitHub 帳號（https://github.com/signup）
- [ ] 在 Mac 上有應用代碼
- [ ] Git 已安裝（`git --version`）

### GitHub 階段
- [ ] 建立 Public Repository
- [ ] 執行 `git init`（已完成）
- [ ] 執行 `git add .`
- [ ] 執行 `git commit -m "Initial commit"`
- [ ] 執行 `git remote add origin <你的URL>`
- [ ] 執行 `git push -u origin main`

### Streamlit Cloud 階段
- [ ] 訪問 https://streamlit.io/cloud
- [ ] 用 GitHub 登入
- [ ] 點擊 New app
- [ ] 選擇 Repository、Branch（main）、File（app.py）
- [ ] 點擊 Deploy
- [ ] 等待 2-3 分鐘部署完成

### 驗收階段
- [ ] 訪問生成的 URL
- [ ] 測試上傳 XLSX 檔案
- [ ] 測試下載 HTML/PDF
- [ ] 確認沒有錯誤訊息
