🚀 5 分鐘快速部署流程
═══════════════════════════════════════════════════════════════════

【第 1 步 - 2 分鐘】建立 GitHub Repository

1. 訪問 https://github.com（如果沒帳號先註冊）
2. 點擊右上角 **+** → **New repository**
3. Repository name: cost-analysis-app
4. 選擇 **Public**
5. 點擊 **Create repository**
6. 複製你的 Repository URL:
   https://github.com/你的用戶名/cost-analysis-app.git


【第 2 步 - 2 分鐘】推送代碼到 GitHub

在你的 Mac 終端執行以下命令（複製貼上即可）：

```bash
cd /Users/grace/Desktop/成本分析

git add .

git commit -m "Cost Analysis App - Initial Upload"

git remote add origin https://github.com/你的用戶名/cost-analysis-app.git

git branch -M main

git push -u origin main
```

⚠️ 注意：替換 URL 中的「你的用戶名」為你的 GitHub 用戶名
提示：如果要查看用戶名，訪問 https://github.com 右上角頭像


【第 3 步 - 1 分鐘】部署到 Streamlit Cloud

1. 訪問 https://streamlit.io/cloud
2. 點擊 **Sign in with GitHub**
3. 授權 Streamlit 訪問你的 GitHub（只需一次）
4. 點擊 **New app**
5. 填入：
   - Repository: 你的用戶名/cost-analysis-app
   - Branch: main
   - Main file path: app.py
6. 點擊 **Deploy**


【自動部署 - 2-3 分鐘】等待部署完成

Streamlit Cloud 會自動：
- 讀取你的代碼
- 安裝依賴
- 啟動應用
- 生成 URL

你會看到：
```
https://cost-analysis-app.streamlit.app
```

⚡ 常見問題秒答
═══════════════════════════════════════════════════════════════════

Q: 需要多少錢？
A: 完全免費！Streamlit Cloud 免費方案無限使用

Q: 應用什麼時候會停？
A: 免費版本 30 分鐘無活動會進入睡眠狀態
   訪問時自動喚醒（5-10 秒）
   付費版本（$5/月）無此限制

Q: 多少人可以同時使用？
A: 無限！取決於伺服器資源

Q: 推送時卡住了？
A: 檢查你的 GitHub URL 是否正確
   執行：git remote -v

Q: 部署失敗怎麼辦？
A: 查看 Streamlit Cloud 日誌
   點擊應用右上角 ⋯ → View logs

Q: 應用很慢？
A: 免費版本首次訪問會較慢（喚醒時間）
   之後訪問速度正常
   升級付費版本可消除睡眠限制

Q: 能用自己的域名嗎？
A: Streamlit Cloud 免費版不支援
   可考慮升級付費版本
   或使用其他平台（Railway 等）

Q: 員工需要 GitHub 帳號嗎？
A: 不需要！員工只需訪問你的 URL
