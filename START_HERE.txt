🔵 步驟 1：建立 GitHub Repository

1. 打開瀏覽器
2. 訪問 https://github.com
3. 登入（沒帳號先註冊）
4. 點擊右上角 + → New repository
5. Repository name: cost-analysis-app
6. 選擇 Public
7. Create repository
8. 複製 URL（類似：https://github.com/你的用戶名/cost-analysis-app.git）


🔵 步驟 2：推送代碼

打開 Mac 終端，複製以下命令（一行一行執行）：

cd /Users/grace/Desktop/成本分析

git add .

git commit -m "Cost Analysis App v1.0"

git remote add origin 替換成你的Repository_URL

git branch -M main

git push -u origin main


⚠️ 替換「替換成你的Repository_URL」為你從步驟 1 複製的 URL
   例如：https://github.com/grace/cost-analysis-app.git


🔵 步驟 3：部署到 Streamlit Cloud

1. 訪問 https://streamlit.io/cloud
2. 用 GitHub 登入
3. 點擊 New app
4. 選擇 Repository: 你的用戶名/cost-analysis-app
5. Branch: main
6. File path: app.py
7. 點擊 Deploy
8. 等待 2-3 分鐘...
9. 得到你的 URL！
